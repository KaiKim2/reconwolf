# ReconWolf

ReconWolf is a Python-based web reconnaissance utility designed for directory and file discovery during authorized security assessments, labs, and CTF environments.

## Features
- Multithreaded scanning
- Custom extension support
- Detects 200, 301, 302, and 403 responses
- Custom User-Agent
- Fast and lightweight
- Simple CLI interface

## Requirements
- Python 3
- requests
- colorama

## Installation
```bash
pip install requests colorama
